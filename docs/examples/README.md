## üéØ Quick Start Examples

### Test a Single File
```bash
# Quick test to see if your setup works
TMDB_API_KEY=your_key plex-media-organizer test "Avengers.Endgame.2019.1080p.BluRay.mkv"

# Expected result:
# ‚úÖ Movie: Avengers: Endgame (2019)
# üìÅ Organized: Avengers Endgame (2019) 1080p BluRay.mkv
```

### Scan a Directory
```bash
# Scan without making changes
plex-media-organizer scan ~/Downloads/movies

# See what would be organized
plex-media-organizer organize ~/Downloads/movies --dry-run
```

### Simple Organization
```bash
# Organize with backup
plex-media-organizer organize ~/Downloads/movies \
    --output ~/Movies/organized \
    --backup ~/backups/organize-backup.json
```

## üîÑ Workflow Examples

### New Movie Collection Workflow
1. **Download** ‚Üí New movies in `~/Downloads/new-movies/`
2. **Test** ‚Üí `plex-media-organizer scan ~/Downloads/new-movies`
3. **Review** ‚Üí Check success rate and failed files
4. **Organize** ‚Üí `plex-media-organizer organize --dry-run` then real organization
5. **Import** ‚Üí Move to Plex library and refresh

### Library Cleanup Workflow
1. **Analyze** ‚Üí Full library scan to understand current state
2. **Plan** ‚Üí Identify problem areas and organization strategy
3. **Backup** ‚Üí Create full backup of current state
4. **Execute** ‚Üí Organize in chunks with frequent backups
5. **Verify** ‚Üí Check results and fix any issues

### Daily Automation Workflow
1. **Watch** ‚Üí Monitor download directory for new files
2. **Wait** ‚Üí Let downloads complete
3. **Process** ‚Üí Automatically organize and move to media server
4. **Notify** ‚Üí Send success/failure notifications
5. **Cleanup** ‚Üí Remove original files and update libraries

## üõ†Ô∏è Configuration Examples

### Light User Configuration
```toml
[apis]
tmdb_api_key = "your_key"

[organization.quality]
preferred_quality = "1080p"
```

### Power User Configuration  
```toml
[apis]
tmdb_api_key = "your_key"

[organization]
output_directory = "/media/organized"
organize_by_type = true

[organization.quality]
preferred_quality = "1080p"
quality_order = ["4K", "2160p", "1080p", "720p"]

[organization.original_titles]
prefer_original_titles = false
include_english_subtitle = false
fallback_to_english_on_error = true

[processing]
max_parallel_files = 16
enable_cache = true
cache_ttl = 7200
```

### Developer Configuration
```toml
[apis]
tmdb_api_key = "your_key"

[logging]
level = "debug"
log_api_calls = true

[processing]
max_parallel_files = 4
enable_cache = false  # Always fresh data
```
